HTML semántico

HTML semántico significa usar las etiquetas correctas según el significado del contenido. Por ejemplo, en lugar de usar solo <div> para todo, se usan etiquetas que describen la función del contenido.

¿Por qué tantos headers?

Cada <header> sirve como cabecera de una página o de una sección. Por eso puede haber varios en un documento, siempre que estén dentro de diferentes secciones o artículos.
